Detailed Description

The ObjectId class is also used in Dagda. Therefore it has the required PortableServer::ObjectId / unbounded sequence interface.

The OID class is 16 bytes, this class is 24 bytes. If one is memory constraint the OID class may be of some help. Be warned: the OID class is not actively maintained anymore (but than, who reeds this besides me, the maintainer).
